<h4>CAI5107: Machine Learning (Fall 2025) @USF</h4>
  <ul>
    <li>Course Instructor: Dr. Anowarul Kabir, Assitant Professor, USF</li>
    <li>Core learning outcomes:
      <ul>
        <li>Understand and explain machine learning algorithms.</li>
        <li>Design a learning algorithm and apply it to a data set.</li>
        <li>Understanding parameter space of learning algorithms to enhance performance.</li>
        <li>Formalizing independent project.</li>
      </ul>
    </li>
    <li>Student Proud Project: A semester-long, independent project where each student will plan and execute a machine learning project from start to finish. The project consists of three big milestones.
    <ul> 
      <li>Milestone I: Literature Survey and Project Proposal</li>
      <li>Milestone II: Reproduction of an Existing Work</li>
      <li>Milestone III: Novel Contributions</li>
      </ul>
    </li>
  </ul>
</div>
